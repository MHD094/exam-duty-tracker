How to Use the Exam Duty Tracker

  1. Select all text from your exam duty pdf or doc (Ctrl + A).
  2. Copy the selected text (Ctrl + C).
  3. Paste the text into the schedule text area below (Ctrl + V).
  4. Then give the short form as input and then press FIND DUTIES button.
